# AlcocheMics Image Generator

## Requirements

- [Rust](https://www.rust-lang.org/)

## Building

```
cargo build --release
```

## Running

```
cargo run --release -- --input examples/green.jpg --background examples/bg.jpg --output out.jpg
```
